Conclusiones - Nicolas Benavides

Hemos podido identificar cómo trabajar bajo la herramienta GitKraken para poder subir un repositorio de forma práctica y sencilla, creación de ramas para distintos entornos de desarrollo por integrante, además de esto poder hacer Commit de los cambios en la página, push y pull para poder subir y bajar cambios a nuestra maquina local, además de esto tuvimos la oportunidad de usar Bootstrap Starter para poder tomar una plantilla en la página Web

Conclusiones - Julián Fuentes

Este trabajo me permitió fortalecer mis conocimientos en control de versiones con Git y GitKraken, aprendiendo la importancia de realizar commits y gestionar repositorios , crear ramas y sincronizarlo con GitHub me ayudó a entender cómo se maneja el flujo de trabajo en el desarrollo en equipo. Además, durante el desarrollo de K-Social, pude aplicar y reforzar mis habilidades en HTML, CSS, JavaScript y Bootstrap.
